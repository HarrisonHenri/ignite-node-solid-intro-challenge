## Install

Clone the repo using

```
$ git clone https://github.com/HarrisonHenri/ignite-node-solid-intro-challenge.git
```

```
$ cd ignite-node-solid-intro-challenge
```

Install all the required dependencies

```
$ yarn
```

Then execute

```
$ yarn dev
```
